Class: Ordered_KVS

core/jsgui-data-structures. Ordered_KVS

Represents an unsorted key/value pairs storage, but with fast access by the key. The each()-based operations (e.g. keys(), values() etc.) returns the items in the same order as they was added. But the get() and out() operations use a fast key access. Unfortunately these operation works with the last added value only if several values have equal keys.

Returns the value for the passed key, or undefined if the key does not exists.
If the key was added several times, then returns the latest added value.

out(key)

Removes the pair with the passed key from the storage. Throws an exception if the key does not exists.
If the key was added several times, then removes the latest added pair only (and throws an exception for the next time).
