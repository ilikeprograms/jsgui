Class: Unique_Constraint

core/constraint. Unique_Constraint

Collection unique constraint.

The class does not provide the match() method, but provides .fields property for the unique field names.

The class public name is Unique (e.g. new Constraint.Unique()).

new Unique_Constraint(spec)

Parameters:
Name Type Description
spec string | Array.<string>

unique field name(s)

Members

fields :string|Array.<string>

Type:
  • string | Array.<string>
Example
var c = new Constraint.Unique("ID");
c.fields  ==> "ID"
